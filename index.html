隐私政策

应用名称：Pet Topics Collector

最后更新：2026-03-05

1. 数据收集

本应用仅收集 Pinterest 公开的 pins（图钉）数据，搜索关键词限于宠物相关（如 cat、dog、pet）。

收集的数据字段包括：

  • Pin 的标题、描述
  • 图片 URL
  • 链接地址
  • 创建时间、板子信息

我们不会收集：

  • 用户个人信息（姓名、邮箱、地址等）
  • 登录凭证或密码
  • 私有 pins 或非公开内容

2. 数据用途

收集的数据仅用于：

  • 市场研究（分析宠物行业趋势话题）
  • 产品开发灵感
  • 内容策略制定

不用于：个性化广告、用户画像销售、第三方分发。

3. 数据存储与安全

  • 数据存储于本地计算机（~/Desktop/宠物线索/）
  • 不传输到远程服务器（除非用户主动选择云同步）
  • 不对外公开或共享
  • 可随时删除本地数据文件

4. 用户权利

由于我们仅处理公开数据，用户的权利主要体现为：

  • 可在 Pinterest 设置中将自己的 pins 设为私有，从而避免被收集
  • 如需删除已收集的数据，请 contacting the developer

5. 第三方服务

本应用使用 Pinterest API（v5）读取公开 pins。使用 Pinterest API 受 Pinterest 开发者条款约束。

Pinterest 隐私政策：https://policy.pinterest.com/en/privacy-policy

6. Cookie 与追踪

本应用不使用 Cookie 或追踪技术。

Privacy Policy

App Name: Pet Topics Collector

Last Updated: March 5, 2026

1. Data Collection

This app collects publicly available Pinterest pins only, with search queries limited to pet-related topics (e.g., cat, dog, pet).

Collected data fields include:

  • Pin title and description
  • Image URL
  • Link URL
  • Creation timestamp and board information

We do NOT collect:

  • Personal information (name, email, address, etc.)
  • Login credentials or passwords
  • Private pins or non-public content

2. Data Usage

Collected data is used solely for:

  • Market research (analyzing trending pet industry topics)
  • Product development inspiration
  • Content strategy planning

Not used for: personalized advertising, selling user profiles, or sharing with third parties.

3. Data Storage & Security

  • Data is stored locally on your computer (~/Desktop/宠物线索/)
  • No transmission to remote servers (unless user opts for cloud sync)
  • No public sharing or distribution
  • Data files can be deleted at any time

4. User Rights

Since we only process public data, user rights primarily include:

  • Setting pins to private in Pinterest settings to prevent collection
  • Requesting deletion of already collected data by contacting the developer

This app uses the Pinterest API (v5) to read public pins. Use of Pinterest API is subject to Pinterest's Developer Terms.

Pinterest Privacy Policy: https://policy.pinterest.com/en/privacy-policy

This app does not use cookies or tracking technologies.
